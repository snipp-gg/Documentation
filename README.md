# Snipp.gg API Documentation

Documentation for the Snipp.gg image and video hosting API.

## Quick links

- [Quickstart](/quickstart) — Get your API key and make your first upload
- [API reference](/api-reference/introduction) — Full endpoint documentation

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview the docs locally:

```bash
npm i -g mint
mint dev
```

View the preview at `http://localhost:3000`.

## Commands

- `mint dev` — Local preview
- `mint broken-links` — Check for broken links
- `mint validate` — Validate the documentation build
